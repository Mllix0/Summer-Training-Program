## Task Summary

### Task 1: Image Recognition Model Using Teachable Machine

This task focuses on training an image recognition model using Google Teachable Machine. The model was trained using three classes: Screwdriver, Wrench, and Pliers. After training and testing, the model was exported in TensorFlow/Keras format and used in a Python script to predict the class of an input image.
