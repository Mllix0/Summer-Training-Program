## About This Repository

Each track contains documentation for the tasks completed during the training program.

The documentation includes:

- Task objective
- Tools used
- Steps followed
- Results
- Challenges
- What I learned
- Images, files, or links when needed

## Documentation Process

For every training task, I will follow this process:

1. Complete the task.
2. Take notes during the task.
3. Collect screenshots, photos, files, or links if needed.
4. Open the correct track page.
5. Update the task table.
6. Add the full task documentation under the Completed Tasks section.
7. Commit the changes to GitHub.
